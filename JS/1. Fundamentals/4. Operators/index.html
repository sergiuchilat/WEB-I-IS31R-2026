Operators

Arithmetic Operators

Used to perform arithmetic calculations.

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus: %
  • Exponentiation: **

Comparison Operators

Used to compare two values.

  • Equal to: ==
  • Not equal to: !=
  • Strict equal to: ===
  • Strict not equal to: !==
  • Greater than: >
  • Less than: <
  • Greater than or equal to: >=
  • Less than or equal to: <=

Logical Operators

Used to combine multiple conditions.

  • Logical AND: &&
  • Logical OR: ||
  • Logical NOT: !
